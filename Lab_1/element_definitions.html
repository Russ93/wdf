This a list of definitions for HTML

Header 1
"<h1>" defines a level 1 heading
Header 2
"<h2>" defines a level 2 heading
Header 3
"<h3>" defines a level 3 heading
Unordered List
"<ul>" an unordered list of items or text
Ordered List
"<ol>" an ordered list of items or text
List Item
"<li>" indicates an individual item in a list
Paragraph
"<p>" indicates a paragraph of text
